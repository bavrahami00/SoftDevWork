Key Features of Bootstrap


Bootstrap is a framework for building responsive, mobile-first sites. It has many features that we will demonstrate below (and above).


There are many ways of organizing information with Bootstrap.

Grid System:

Containers, rows, and columns together can be used to create a grid system
Columns are automatically divided equally like this.
But don't have to be!
Borders can also be added to said columns
There can be a max of 12 columns per row.

Other tools available for use with grids are offsetting some columns to the right (as shown on the second row), changing the margins and gutters, and nesting rows inside each other.

Tables:

Tables are similar to grids. They cannot be as flexible about the placement and relative width of entries, but they automatically provide borders and headings, among other features.
